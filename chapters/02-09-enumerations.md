## Enumerations

:::::::::::::: {.columns align=center}
::: {.column width="55%" }

* Since version 8.1
* Enumerations are not extendable.
* Enumerations can have methods.
* Enumerations can have backed values.
* Enumerations can implement interfaces.
* Enumerations can use traits.
